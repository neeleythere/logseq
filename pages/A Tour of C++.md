- ### Initialization #cpp 
  Prefer curly-brace initialization, you can even continue using the `=` sign:
  ```cpp
  double d1 {2.3};
  double d2 = {2.3}; // both good
  ```
- Use `auto` when there is not a specific reason to mention the type. Wanting to be clear is a valid reason.
